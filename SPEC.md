# 成华县过县 - 局域网即时通讯软件

## 1. 项目概述

**项目名称**：成华县过县
**类型**：桌面应用（跨平台）
**技术栈**：Tauri 2.x + React + TypeScript
**目标**：在局域网内实现 Mac 与 Windows 之间的即时通讯和文件传输

---

## 2. 功能需求

### 2.1 设备发现
- 自动发现同一局域网内的在线用户
- 显示设备名称、IP、在线状态
- 心跳机制检测设备在线/离线

### 2.2 即时通讯
- 点对点文字消息发送
- 消息实时送达
- 消息历史记录（本地存储）

### 2.3 文件传输
- 支持任意文件格式
- 显示传输进度
- 支持大文件断点续传（可选，V1 先不做）

---

## 3. 技术架构

### 3.1 通信协议
- **设备发现**：UDP 广播（端口 19876）
- **消息传输**：TCP 长连接（端口 19877）
- **文件传输**：TCP 流式传输（端口 19878）

### 3.2 消息格式
```json
{
  "type": "message|file|heartbeat|offline",
  "from": "设备ID",
  "fromName": "设备名称",
  "to": "目标设备ID（广播时为空）",
  "content": "消息内容",
  "timestamp": 1234567890,
  "fileName": "文件名（文件传输时）",
  "fileSize": 12345（字节）
}
```

## 4. UI 设计

### 4.1 界面布局
- **左侧栏**：设备列表（在线用户）
- **右侧主区域**：聊天窗口 / 文件传输窗口（Tab 切换）
- **顶部**：当前会话用户信息

### 4.2 视觉风格
- 扁平化设计
- 配色沿用「杨小乖成绩考核表」的游戏五色系
- 简洁干净，专注功能
